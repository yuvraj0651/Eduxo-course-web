# 📘 Eduxo — Course Website

<h3 align="center">A modern, interactive course platform UI built with React.js and Tailwind CSS, designed to showcase online courses with smooth UI and responsive layout.</h3>

---

## 🚀 Overview
**Eduxo** is a frontend course platform UI that demonstrates modern frontend design and development skills using **React.js** and **Tailwind CSS**. This project focuses on creating a **responsive, interactive interface** that can be extended for real course platforms in the future.

It’s ideal for showcasing understanding of:
- React components & JSX
- Responsive UI design with Tailwind
- Navigation, cards, modals, and layout structure
- Interactive user interface elements

---

## 🧠 Features
- 🎨 Clean & modern UI for a course platform
- 💻 Responsive layout for desktop & mobile
- 🧱 Modular React component structure
- 📚 Course card listing with hover effects
- ⚡ Built with Tailwind CSS for easy styling

---

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS, HTML5, CSS3
- **Languages:** JavaScript (ES6+)
- **Tools:** Git, GitHub
